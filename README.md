\# CBT App



A Python console-based Computer Based Test (CBT) application with MySQL database.



\## Features

\- User registration and login

\- Multiple choice questions

\- Timed examinations

\- Score tracking and results

\- Admin dashboard for managing questions and users



\## Requirements

\- Python

\- MySQL

\- XAMPP



\## Installation



1\. Clone the repository:

 git clone https://github.com/agboolataiwo121/cbtapp.git



2\. Install dependencies:

pip install mysql-connector-python python-dotenv



3\. Start your MySQL server (XAMPP)



4\. Run the app:

python cbtapp.py
